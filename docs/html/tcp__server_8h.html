Detailed Description

This is a modified version of esp-idf/examples/protocols/sockets/tcp_server. Choose IPv4, IPv6 or both via setting EXAMPLE_IPV4 and EXAMPLE_IPV6, set server port number with EXAMPLE_PORT, keep-alive idle time, interval time and package resend count with EXAMPLE_KEEPALIVE_IDLE, EXAMPLE_KEEPALIVE_INTERVAL and EXAMPLE_KEEPALIVE_COUNT under TCP Server Configuration submenu in project configuration menu.
